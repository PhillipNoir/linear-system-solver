Realiza la eliminación hacia atrás y normalización de pivotes en la matriz A.

Esta función convierte A en una matriz identidad (forma reducida por filas) mediante eliminación hacia atrás y normalización de pivotes. Es el paso final del método de Gauss-Jordan.

Parameters
AMatriz de coeficientes (modificada).
bVector columna de términos independientes (modificado).
mostrarPasosSi es verdadero, imprime los pasos si el sistema es pequeño.
Exceptions
std::runtime_errorSi se detectan pivotes demasiado pequeños o inconsistencias numéricas.

◆ forwardElimination()

void forwardElimination ( Matrix & A,
Matrix & b,
bool mostrarPasos )

Realiza la eliminación hacia adelante (triangulación) del sistema Ax = b.

Esta función aplica el método de eliminación de Gauss para convertir la matriz A en una forma triangular superior, modificando también el vector columna b. Detecta sistemas inconsistentes o numéricamente inestables.

Parameters
AMatriz de coeficientes (modificada en el proceso).
bVector columna de términos independientes (modificado en el proceso).
mostrarPasosSi es verdadero, imprime cada paso si el sistema es pequeño (≤ 10 ecuaciones).
Exceptions
std::runtime_errorSi el sistema es numéricamente inestable o inconsistente.
