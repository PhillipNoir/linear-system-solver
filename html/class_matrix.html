Detailed Description

Clase que representa una matriz dinámica de tamaño variable.

Esta clase permite crear matrices de tamaño definido en tiempo de ejecución, acceder y modificar sus valores mediante el método 'at', y realizar operaciones básicas, como impresion y llenado manual. Utiliza punteros inteligentes (std::unique_ptr) para gestionar la memoria de forma segura.
