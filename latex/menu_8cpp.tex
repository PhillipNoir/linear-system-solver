\doxysubsection{Detailed Description}
Muestra el menú principal del programa y gestiona la interacción con el usuario. 

Permite al usuario resolver sistemas de ecuaciones lineales utilizando los métodos de\+:
\begin{DoxyItemize}
\item Eliminación de Gauss
\item Eliminación de Gauss-\/\+Jordan
\end{DoxyItemize}

El usuario puede\+:
\begin{DoxyItemize}
\item Ingresar el tamaño del sistema
\item Proporcionar los coeficientes (matriz A) y el vector independiente (b)
\item Elegir si desea ver los pasos intermedios de la solución
\item Ver el resultado del sistema resuelto
\item Salir del programa
\end{DoxyItemize}

La función se ejecuta en un bucle hasta que el usuario selecciona la opción de salir.
