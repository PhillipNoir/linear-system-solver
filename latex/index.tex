Solución de sistemas lineales por Gaus y Gauss-\/\+Jordan en C++. Linear system solver using Gaussian methods

Este es un programa de consola en {\bfseries{C++ puro}} que permite resolver sistemas de ecuaciones lineales utilizando los métodos de {\bfseries{eliminación de Gauss}} y {\bfseries{eliminación Gauss-\/\+Jordan}}. Diseñado como una herramienta educativa y funcional, es ideal para estudiantes de ingeniería, álgebra lineal o métodos numéricos.

\hypertarget{index_autotoc_md7}{}\doxysubsubsection{\texorpdfstring{1. Ejecutar}{1. Ejecutar}}\label{index_autotoc_md7}
El programa se ejecuta en consola. Al iniciarse, le pedirá\+:


\begin{DoxyItemize}
\item El método por el que deseas resolver tu sistema.
\item La dimensión del sistema.
\item Ingresar los datos de la matriz A (el mismo programa indicará la posición de la matriz que está ingresando).
\item Ingresar los datos del vector b.
\item Pregunta si desea que se muestren los pasos de la solución (ingresar 1 para si, 0 para no).
\item Puede hacer cuantos sistemas desee, para salir debe seleccionar la opción 3.
