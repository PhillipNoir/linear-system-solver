\doxysubsection{Detailed Description}
Clase que representa una matriz dinámica de tamaño variable. 

Esta clase permite crear matrices de tamaño definido en tiempo de ejecución, acceder y modificar sus valores mediante el método \textquotesingle{}at\textquotesingle{}, y realizar operaciones básicas, como impresion y llenado manual. Utiliza punteros inteligentes ({\ttfamily std\+::unique\+\_\+ptr}) para gestionar la memoria de forma segura.
