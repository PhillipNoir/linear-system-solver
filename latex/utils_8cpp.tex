Realiza la eliminación hacia adelante (triangulación) del sistema Ax = b. 

Esta función aplica el método de eliminación de Gauss para convertir la matriz A en una forma triangular superior, modificando también el vector columna b. Detecta sistemas inconsistentes o numéricamente inestables.
