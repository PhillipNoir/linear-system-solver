Resuelve un sistema de ecuaciones lineales mediante el método de eliminación de Gauss con pivoteo parcial.

Este método transforma la matriz A en una matriz triangular superior utilizando eliminación hacia adelante, luego resuelve el sistema triangular resultante mediante sustitución regresiva.

Parameters
AMatriz de coeficientes del sistema (modificada durante la ejecución).
bVector columna de términos independientes (modificado durante la ejecución).
Returns
std::vector<double> Vector solución del sistema.
Exceptions
std::runtime_errorSi el sistema no tiene solución única (pivote cero en la diagonal).

Resuelve un sistema de ecuaciones lineales mediante el método de Gauss-Jordan.

Este método transforma la matriz A en una matriz diagonal utilizando eliminación hacia adelante y atrás, lo que nos entrega el vector solución de forma directa.

Imprime el sistema de ecuaciones lineales en forma matricial.

Imprime un sistema de ecuaciones lineales en forma matricial.

Esta función muestra en consola la matriz de coeficientes A y el vector de constantes b en forma de sistema ampliado (matriz aumentada), alineando los valores para mayor legibilidad.

Por ejemplo, imprime una salida como esta:

2 -1 3 | 9
1 0 -2 | 3
Parameters
AMatriz de coeficientes del sistema de ecuaciones (tipo Matrix).
bVector columna con los términos independientes (tipo Matrix).
